1. Use the Documentation button to navigate to the Documentation page or use 'localhost:10533/documentation.html'
  2. Use localhost:10533/movies to display the whole movie list
  3. Use /movies/:title (Replace :title with movie name) to show movie from array
  4. Use /genre/:genreName (Replace :genreName with wanted genre) to show movie with this genre from array
  5. Use /directors/:directorName (Replace :directorName with Director name) to show movie with the director from array
  6. /users/:id (Replace :id with numbers 1 or 2) to show user info in postman
  7. /users/:id/favorateTitle (Replace :favorateTitle with movie name) to add users favorateTitle to there info in postman
  8. /users/:id/:update (Replace :id with) user id in postman and update with new username to update info
  9. /users/:id/:delete (Replace :id with user id) in postman to delete user from records
